## Quality Gate Reference

Before completing any milestone, verify:

- [ ] **Brand** — Does it match Brand Bible (Manual 01)?
- [ ] **Product narrative** — Does it communicate the right outcome (Manual 02)?
- [ ] **Visual consistency** — Does it use design tokens only (Manual 03)?
- [ ] **Technical standards** — Does it meet architecture rules (Manual 04)?
- [ ] **Experience integrity** — Does it support the scene story (Manual 05)?

All five must pass. No exceptions.

## Documentation Governance

**One idea, one location.** Every concept exists exactly once. Other documents reference rather than repeat.

Before adding content to any manual:
1. Search this index — does the concept already exist?
2. If yes — add a cross-reference, not a copy
3. If no — add to the correct manual, then update this index

**The documentation must outlive the current website.** It is the operating system of the PYRAXIS brand.
